# Uber Clone

A mobile app that recreates the core Uber experience - request rides, track drivers, and get from point A to point B. Built as a learning project to explore Android development with Kotlin.

## What it does

This app lets you experience the rider side of a ride-sharing service:

- Sign up with your phone number or social media accounts
- See your location on a map and request a ride
- Track your driver's location in real-time
- View your ride history and trip details
- Rate your experience after each trip
- Manage your profile and preferences

## About this project

This is a demo application created to learn Kotlin and Android development. It mimics the user interface and basic functionality of popular ride-sharing apps like Uber. The app includes user authentication, maps integration, and a clean interface for booking rides.

## Getting started

To run this project, you'll need to set up a few things:

1. Get a Google Maps API key for the map features
2. Set up Firebase for user accounts and data storage
3. Configure social login if you want Facebook/Google sign-in

The app is designed for Android phones and requires Android 6.0 or newer.
